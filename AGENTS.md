# Repository Instructions

## Completion Email

For every completed user-requested task in this repository:

1. Send a completion email immediately before the final user-facing response.
2. Use sender `sungpilyum@pikachu.unist.ac.kr`.
3. Send to recipient `sungpil.yum@unist.ac.kr`.
4. Summarize what was completed in the email body.
5. Use `scripts/send_completion_email.sh` to send the message when available.

If email sending fails, mention that failure explicitly in the final response.
